Overview

Encapsulates a server-side HTTP request.

An instance of this class is created for each request that is handled by the server and is passed to the user via the handler specified using Nodex::HttpServer#request_handler.

On creation a new execution context is assigned to each instance and an event loop is allocated to it from one of the available ones. The instance must only be called from that event loop.

Each instance of this class is associated with a corresponding HttpServerResponse instance via the field #response.

def header_names
  if (@header_names == nil)
    names = @j_del.getHeaderNames
    iter = names.iterator
    @header_names = Set.new
    while iter.hasNext
      name = iter.next
      @header_names.add(name)
    end
  end
  @header_names
end

def headers
  if (@headers == nil)
    hdrs = @j_del.getHeaders
    iter = hdrs.entrySet.iterator
    @headers = {}
    while iter.hasNext
      entry = iter.next
      @headers[entry.getKey] = entry.getValue
    end
  end
  @headers
end
