Overview

Encapsulates a client-side HTTP response.

An instance of this class is provided to the user via a handler that was specified when one of the HTTP method operations, or the generic Vertx::HttpClient#request method was called on an instance of HttpClient.

def header_names
  if @header_names == nil
    names = @j_del.getHeaderNames
    iter = names.iterator
    @header_names = Set.new
    while iter.hasNext
      name = iter.next
      @header_names.add(name)
    end
  end
  @header_names
end

def headers
  if @headers == nil
    hdrs = @j_del.getAllHeaders
    iter = hdrs.entrySet.iterator
    @headers = {}
    while iter.hasNext
      entry = iter.next
      @headers[entry.getKey] = entry.getValue
    end
  end
  @headers
end

- (Object) trailer(key)

Returns a trailer value, or nil if no such trailer exists. Trailers will only be available in the response if the server has sent a HTTP chunked response where headers have been inserted by the server on the last chunk. In such a case they won’t be available on the client until the last chunk has been received.
